🧙 Instalación guiada

Al instalar el módulo, en lugar de aplicar valores fijos, se presenta un formulario de configuración inicial que el administrador completa antes de que el sitio empiece a funcionar. Esto permite reutilizar el módulo en cualquier proyecto sin estar atado a nombres específicos.

Ajuste configurable en la instalaciónValor por defecto
Nombre del tipo de contenido principal Inmueble
Nombre del tipo de contenido de agentes Agente
Nombre del tipo de contenido de documentos Documentos
Prefijo del código automático PROP
Separador del código -
Número de dígitos del código 5
💡 Implementación: hook_install_tasks() en property_manager.install define una tarea de configuración inicial que Drupal ejecuta automáticamente al terminar la instalación del módulo. El formulario redirige al administrador antes de que pueda usar el sitio.
✅ Decisión: Instalación guiada. Los valores por defecto permiten instalar y usar sin necesidad de configurar, pero el formulario está disponible si se quiere personalizar. Ver Campo Código →
