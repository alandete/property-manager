Fase 5 — Features adicionales

Fase 5: Features adicionales

Sub-módulos independientes que se activan según las necesidades de cada instalación. Ninguno es obligatorio para que el sitio funcione. Se priorizan según la demanda real del proyecto.

⏳ Estado: Pendiente. Requiere completar la Fase 1. El resto de fases previas mejoran la experiencia pero no son bloqueantes para estas features.

🧩 Sub-módulos opcionales

Sub-móduloFeatureDependencias
property_manager_seo Schema.org / JSON-LD para inmuebles. Tipo RealEstateListing. Mejora snippets en Google. composer require drupal/schemadotorg
property_manager_pdf Genera una ficha PDF descargable de cada inmueble con todos sus datos y fotos. composer require drupal/entity_print tecnickcom/tcpdf
property_manager_contact Formulario "Me interesa este inmueble" y formulario de contacto del agente. Envío por correo. composer require drupal/webform
property_manager_workflow Estados del inmueble: Disponible → En negociación → Arrendado/Vendido → Archivado. Módulo core content_moderation + workflows
